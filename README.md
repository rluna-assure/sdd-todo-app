### Todo App
This is an app for creating boards and tasks.
It Allows create multiple tasks in different boards.
	1. Each task can move from open, in-progress to close
	2. Is possible Add comments where task is in-progress
	
[[00_initial_requeriments]]

- 1 Run by CMD
	##### Run
	dotnet run
	#### Tests
	dotnet test

- 2 Run in VS
	Open project in Visual Studio and run the application in debug mode.
